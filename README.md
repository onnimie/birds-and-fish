# birds-and-fish
A simulation for a group of single-type-objects. These objects will follow an algorithm that makes the whole group seemingly act like a hive-minded entity. Each single bird should follow the same algorithm, that does, however, take into account for each individuals velocity, direction and position (neighbours). This should make the whole group act very similarly but without perfect harmony - hopefully creating an interesting simulation for a group of birds (or fish, if you wish).

The simulation is ran in Unity

There are two colours for the birds so I might add a two-party system to the simulation, we'll see.
